# Patch 10：市场方向系统（年份随机 + 方向选择 + 行业旁白）

---

## 概述

开局内部随机抽取一个"市场年份"，决定当前局的热门赛道。
玩家感知年份靠事件旁白，不直接告知。
月1-2玩家选定产品方向，方向与年份的匹配度影响最终结局判定（接入patch 09门槛3）。

---

## 新增游戏状态字段

```javascript
marketYear: number,        // 内部随机，2010-2026，不显示给玩家
gameDirection: null,       // string | null，月1-2锁定后填入
directionChosen: false,    // boolean，是否已完成方向选择
```

## 公司规模 → 方向选择过滤

方向选择事件中，根据开局公司规模卡过滤可显示的选项：

| 公司规模 | 可出现的方向 |
|---------|------------|
| 大厂 | OPENWORLD / ARPG / MOBA / SLG |
| 中型 | CARD / ARPG / MOBA / ROMANCE / IDLE |
| 小厂 | CASUAL / CARD / IDLE / PARTY / ROMANCE |

从该规模的可用方向中，随机抽3个展示给玩家。
若当年热门方向不在该规模的可用列表中，玩家本局天然处于方向劣势（不提示）。

---

## 事件一：行业动态旁白

**类型**：无选项轻量提示，点击关闭
**触发**：每月随机选1-2周，在 event 阶段触发（不替换主事件，作为附加信息条出现）
**条件**：每条旁白在同一局只触发一次

每个 marketYear 配备2条旁白文案（见下方文案表）。

---

## 事件二：方向选择

**触发**：月1 week2，若 directionChosen = false 则强插入 event 阶段
**拖延惩罚**：
- 若 week4 结束仍未选择：每周触发老板干预事件概率+20%，qualityDebt每周+3
- 若月2结束仍未选择：gameDirection 随机赋值（系统强制），玩家失去选择权

**事件格式**：
```
背景文：2-3句（见各年份文案）
选项：3个方向（根据公司规模过滤后随机抽取）
每个选项显示：[方向名] + 一句pitch flavor（见文案表）
```

选定后：`gameDirection` 锁定，`directionChosen = true`，无即时数值效果。

---

## 事件三：同行质疑

**触发**：月2-3，仅当 `gameDirection === mockedDirection[marketYear]` 时触发，每局最多一次
**效果**：叙事事件，选项影响士气/信任度（数值小，重在叙事）

---

## 胜利判定接入（patch 09 门槛3）

```javascript
// 在 patch 09 门槛2（qualityDebt >= 80）之后判断：

const hotDirections = YEAR_DATA[marketYear].hot
const isMatch = hotDirections.includes(gameDirection)
const isSecondaryMatch = hotDirections.indexOf(gameDirection) === 1

if (!isMatch) {
  → 结局「叫好不叫座」
} else {
  → 继续门槛4判断（综合分）
}
```

次热方向（secondary）命中时，综合分阈值从45降为35（容错一点）。

## 文案表

### 2013（CARD年）

**旁白A**
隔壁组策划下班时说了句话："我们月活破百万了，就靠加了个SSR保底。"你点点头，好像听懂了。

**旁白B**
产品会上有人提了个词："抽卡钩子。"没有人问这是什么。大家都在点头。

**方向选择背景文**
立项评审前一天。市场上什么都有人做，但不是什么都有人买单。你需要定下方向。

**方向pitch文案（本年可用）**
- CARD：「低门槛，高频付费，玩家上手快。」
- ARPG：「差异化竞争，走精品路线。」
- SLG：「长线留存，但慢热。」

**同行质疑（SLG被嘲笑）**
> 月度行业沙龙。一个做了三年卡牌的制作人看着你："SLG？手机玩家坐不住的。"他不是在问，是在陈述。

- A：「我们做的不一样。」→ 士气+3
- B：微笑，换话题。→ 无效果
- C：「你说得对，我们在重新评估。」→ 士气-5，老板压方向事件多一个选项

---

### 2017（吃鸡+女性向）

**旁白A**
公司楼道里有人在玩一个游戏，跳伞，捡枪。他在电话里说："大吉大利，今晚吃鸡。"你不知道那是什么意思，但你记住了这句话。

**旁白B**
产品部发来一份报告：女性用户付费意愿同比提升240%。末尾附了一句："女性向蓝海，现在入场还不晚。"没人知道这个240%是怎么算出来的。

**方向选择背景文**
风口这个词今年说烂了，但人人都在找那扇窗。你得定下来。

**方向pitch文案（本年可用）**
- MOBA：「三分天下，你来做第四个。」
- ROMANCE：「新蓝海，先入场先吃肉。」
- IDLE：「不烧钱不烧脑，低成本稳健。」

*(本年无被嘲笑方向事件)*

---

### 2021（原神震荡年）

**旁白A**
原神的数据还在刷屏。一个从业十五年的制作人在朋友圈发了一条："我们是不是都理解错游戏了。"点赞218，评论关闭。

**旁白B**
老板发来一张截图，是某游戏的首周流水数字，旁边附了一个问号。你不知道怎么回复，先点了个已读。

**方向选择背景文**
这一年，有一款游戏改变了所有人对手游的理解。你站在原地，需要决定：跟，还是不跟。

**方向pitch文案（本年可用）**
- OPENWORLD：「标杆已立，跟不上就出局。」
- ROMANCE：「垂直赛道，和那款游戏不正面竞争。」
- CARD：「风险可控，但你知道市场在嫌弃你。」

**同行质疑（CARD被嘲笑）**
> 行业群里有人发了张表情包，一个老头在说"别人家孩子都在做开放世界"。下面有人@你："哥们儿还在做卡牌？"

- A：「精耕细作，垂直深度。」→ 无效果
- B：不回复，退出群聊。→ 士气-3
- C：「是啊，专注做好一件事。」→ 士气+2

---

### 2023（AI浪潮年）

**旁白A**
技术群里有人发了一张AI生成的原画，跟美术组画的差不多。没有人说话。两分钟后，美术主管发了一个微笑表情。

**旁白B**
CEO在全员大会上说了一个词：AI原生。他重复了三次。会后有人悄悄问你："什么是AI原生？"你说你也不知道。

**方向选择背景文**
今年没有人在讨论玩法，所有人在讨论技术。你需要决定，这件事跟你做的游戏有没有关系。

**方向pitch文案（本年可用）**
- AI_NATIVE：「赌对了，未来三年吃红利。」
- PARTY：「UGC是流量密码，已经有人证明了。」
- OPENWORLD：「大力出奇迹，但钱要够烧。」

**同行质疑（OPENWORLD被嘲笑）**
> 沙龙上，主持人介绍你时说："这位在做开放世界。"台下有人小声说了什么，你没听清，但旁边的人笑了。

- A：站起来讲差异化思路。→ 信任度+1
- B：当作没听见。→ 无效果
- C：下课后找到那个人，问他笑什么。→ 士气-3，解锁隐藏方向建议
