# Patch 15：跨行业第五张卡

## 问题

开局选「跨行业」后直接进游戏，6个子行业的差异化 buff/debuff 未实现。

---

## 新增游戏状态字段

```javascript
industryBackground: null  // string | null，仅跨行业玩家有值
```

---

## 交互流程

```
第4张卡选「跨行业」
  → 触发第5张卡「你从哪个行业来？」
  → 玩家选择子行业
  → 揭示效果文案
  → 进入游戏
```

第5张卡样式与其他卡牌组一致，标题：「你从哪个行业来？」

---

## 六个子行业

### 🏠 地产
- **reveal**：「你懂得如何和甲方谈钱。游戏行业的人，谈判能力普遍让你失望。」
- **效果**：
  - 紧急融资行动效果 ×1.5（融资金额上限提升）
  - 预算谈判类事件解锁额外选项（内容填充阶段实现）
  - debuff：前3个月，团队对你的技术判断天然存疑，技术相关事件多一个「团队沉默」结果

### 📚 教培
- **reveal**：「你擅长激励人，但你低估了游戏行业有多混乱。」
- **效果**：
  - 团队关怀行动效果 ×1.3
  - debuff：进度估算行动（月底milestone评估）偏乐观，qualityDebt积累速度 ×1.1

### 💰 金融
- **reveal**：「数字对你来说是母语。团队觉得你不像做游戏的。」
- **效果**：
  - 预算数值对玩家完全透明（加班费系数、预算消耗精确显示）
  - 紧急融资行动成功率 +20%
  - debuff：团队关怀行动效果 ×0.7（你讲道理，但团队需要的不是道理）

### 🎬 影视
- **reveal**：「你见过更大的烂摊子。这里的混乱，你不陌生。」
- **效果**：
  - 幻想家事件 / 空中楼阁事件：解锁专属第四选项（内容填充阶段实现）
  - 市场热潮事件中「PV方向」类选项抗性 +1（debuff减半）
  - debuff：容易被空中楼阁事件的选项A吸引（选A时额外 qualityDebt+5）

### ⛓️ 区块链/Web3
- **reveal**：「你有一套独特的融资话术。团队第一周就在背后议论你。」
- **效果**：
  - 紧急融资行动效果 ×2.0（融资能力最强）
  - 预算类谈判事件额外选项（内容填充阶段实现）
  - debuff：初始士气-10（团队对你天然不信任）；团队关怀行动效果 ×0.6

### 📱 MCN/直播
- **reveal**：「你懂流量，懂老板要什么。但你完全不懂这个东西怎么做出来。」
- **效果**：
  - 向上管理行动效果 ×1.5（每次信任度恢复更快）
  - debuff：前2个月，技术类事件（屎山/完美主义/远见）的选项效果偏差 ×1.2（你的判断不准）

---

## 效果接入方式

stat 类修改（行动效果系数）：在行动执行函数中读取 `industryBackground` 做系数调整

debuff 类修改（qualityDebt 额外积累）：在对应事件选项结算时，检查 `industryBackground` 后叠加

事件专属选项（影视/地产/区块链）：内容填充阶段接入，本 patch 暂不实现，留字段占位
