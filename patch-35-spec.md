## 二、生成逻辑

```js
function generateCharacterName(usedNames) {
  const available = NAME_POOL.filter(n => !usedNames.includes(n));
  return available[Math.floor(Math.random() * available.length)];
}
```

开局生成角色池时，依次调用，保证同局不重名。
