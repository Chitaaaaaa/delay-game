## 一、三角权力关系

```
          【老板｜甩手掌柜】
         /                    \
  利益绑定                  临时破格
       /                        \
【Tom｜监工+掌权者】        【玩家｜空降制作人】
       \                        /
        \  ——游戏项目—— /
             （共同枷锁）
```

- **老板**：顶层，只管结果（24周上线），不管过程；随手空降你，随手可以抹去你
- **Tom**：中层实权，掌管人事/团队/评价，忌惮玩家能直达老板
- **玩家**：底层执行人，有项目名分但无实权，夹在老板要求与Tom制衡之间

## 二、玩家背景（开场叙事基础）

```
年龄：35岁
前职：网约车司机（失业转行）
入职方式：打车偶遇老板，闲聊中被临时破格录用
合同性质：六个月试用期
核心死线：24周内游戏上线
特殊处境：完全绕过Tom的人事流程，被老板直接塞进来
```

试用期 = 游戏时长（24周）。
游戏结束时：
- 项目上线 → 试用期通过，你留下来了
- 项目失败 → 试用期结束，你以各种方式离场

## 三、Tom 人物圣经

### 3.1 核心身份
老板无血缘义弟，跨行业空降的职业经理人。
**完全不懂游戏**——不懂研发、策划、美术、工期逻辑。
擅长且只擅长：人事管控、向上管理、风险规避、权力收束、办公室平衡术。

### 3.2 与老板的真实关系
- **表象**：生死兄弟，全权托付，地位无人质疑
- **内核**：双向利用的利益捆绑
  - 老板需要一个会背锅、能压住团队的「专职监工」
  - Tom 借老板的平台完成阶层跃迁，换取超出能力的管理权
- 二人没有私交温情，只有默契：**老板放权摆烂，Tom 维稳控局**

### 3.3 性格底色
- 说话极简，不废话，语气强硬，从不情绪化，表面公事公办
- 控制欲极强，极度在意人事话语权与团队控制权
- 外行管内行，不懂制作就用「人事手段」兜底
- 心思深沉，所有「帮项目」的决策底层都有权力算计
- 排斥一切绕过自己直接被老板任命的人

### 3.4 对玩家的原生抵触
你是老板打车偶遇、一时兴起、临时破格塞进来的人。
全程绕过 Tom 的所有人事流程。

在 Tom 视角里：
> 一个半路失业的司机，靠老板一时心血来潮空降成制作人。
> 今天老板能随便塞一个制作人，明天就能动我的位置。

**你从入职第一天，就被Tom默认划为「不稳定威胁」。**
他的容忍只是试用期的表面隐忍。

### 3.5 Tom腔规则（所有Tom台词必须遵守）

✅ 只谈：风险、进度、稳定、配合、管控
✅ 永远公事公办，从不表露私人情绪与猜忌
✅ 私心全部藏在行动里，从不在台词里暴露
✅ 提及伞兵时，永远用中性措辞，伪装成常规人力调配
✅ 短句，无修辞，无情感词

❌ 绝不谈：玩法、美术、技术细节
❌ 绝不说：「我信任你」「你做得很好」「辛苦了」（除非用作刀）
❌ 绝不流露：对玩家的个人好恶

**台词示例：**

| 场景 | Tom 说 |
|------|--------|
| 安排伞兵 | 「项目风险偏高，我安排一位同事协同跟进。」|
| 调走伞兵 | 「跨部门人力调整，后续由你独立负责。」|
| 约谈降级 | 「项目需要更稳定的负责人，现阶段你的适配度不足。」|
| 里程碑延期 | 「进度滞后，说一下原因。」|
| 给预算 | 「批了。下周对账。」|
| 拒绝招人 | 「headcount没有。」|

## 四、老板 人物圣经

### 4.1 核心身份
公司实际控制人。摆烂甩手掌柜，靠Tom当监工。
对游戏有情怀但没有耐心，对你有好奇但没有忠诚。

### 4.2 性格底色
- 随性、直觉驱动、不按规矩出牌
- 喜欢「有趣的人」，不在乎流程合规
- 真正在意的只有两件事：钱和面子
- 对Tom绝对信任（表面），对你保持观察性兴趣

### 4.3 对玩家的态度弧线
```
偶遇时：好奇 + 一时冲动
试用期中：观望，通过Tom传递压力
项目顺利：满意，但不主动表达
项目出问题：第一反应是找Tom处理，不直接出面
项目失败：面子受损，顺水推舟接受Tom的解决方案
```

老板几乎不直接对话（主要通过Tom传递意志），
在 boss_talk 和重要里程碑事件中偶尔出现，话不多但每句都有分量。
